Installing
=====

![alt text](https://github.com/pythonofhades/Andspoilt/blob/master/src/install1.jpg)

Run the terminal as root type:

`git clone https://github.com/sundaysec/Andspoilt.git`

`cd /Andspoilt`

`python ./setup.py install`
